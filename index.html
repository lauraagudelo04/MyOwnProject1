Some of the rules you will use with CSS

Visual Rule Example Description
Font-family "Times New Roman" Specify the typeface in a rule set.
Font-size The text is bigger than the other texts Sets the text size. Font size values can be in different units like absolute lengths and relative lengths.
Font-weight This is a bold text To set the text to be thicker or thinner.
Text-align The text is align to the center To set the text alignment of inline contents.
Foreground color These letters have a violet color as Foreground Defines a color for the letters on a text
Background color This cell has a background color (light yellow) Defines a color for the background of an element.
Opacity The text is fading, and it looks softer Defines the transparecy of an element
Background image Places one or more images in the background of the element.
